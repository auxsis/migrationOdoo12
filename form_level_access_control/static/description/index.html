Form Level Access Control

Keep your forms more secure

  • Main objective of the module is that every Form in the Odoo can be made more secure even if the user is is authenticated at the Odoo login.
  • The security can be configured by the users under the Administration/Settings Group.
  • Global Password is defined in the configuration form applied at the authentication wizard.

Configuration

Configure the Security

A configuration is provided under the Settings -> Form Level Access Control -> Form Level Access Control, where the Users/User Groups, Password, Menus are configured.

Authentication before the access

1. The Users defined under the configured group are authenticated(A wizard popped out to enter the Password) before they can able to access the form.

2. Access is granted only on the successful authentication. On failure, it displays a warning.
